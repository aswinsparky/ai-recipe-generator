# AI Recipe Generator

This project is an AI-powered recipe generator built using React, TypeScript, and Vite.

## Features
- Enter a list of ingredients, and the app suggests possible recipes.
- Uses AI to generate unique and creative recipe ideas.
- Built with React + TypeScript + Vite for a fast and efficient development experience.

## Getting Started
### Installation
```sh
git clone https://github.com/aswinsparky/ai-recipe-generator.git
cd ai-recipe-generator
npm install
npm run dev
